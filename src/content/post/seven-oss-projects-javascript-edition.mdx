## Appwrite

<a href="https://github.com/appwrite/appwrite" class="link">Appwrite</a> is an end-to-end backend server for Web, Mobile, Native, or Backend apps.
You can easily integrate your app with user authentication, a database for storing
and querying users and team data, storage and file management, image manipulation,
Cloud Functions, and more services.

## melonJS

<a href="https://github.com/melonjs/melonJS" class="link">melonJS</a> is an HTML5 game engine that empowers developers and designers to focus on content.
The framework provides a comprehensive collection of components and support for a number of third-party tools.
It is an alternative to other JavaScript game engines such as <a href="https://phaser.io/" class="link">Phaser</a> ü§û

## GDevelop

<a href="https://github.com/4ian/GDevelop" class="link">GDevelop</a> is another free, open-source game engine build from any platforms like PC or phone.
The peculiarity of this game engine is that it is possible to create video games without using code, because it uses a logic system based on conditions and actions.

## Egg.js

<a href="https://github.com/mikeflynn/egg.js" class="link">Egg.js</a> is a simple library that allows you to easily add web easter eggs by watching the user's key strokes.
It's simple to use and very funny ü§£

## API Vault

<a href="https://github.com/Exifly/ApiVault" class="link">API Vault</a> is a beautiful project written in JavaScript: it's an online platform that works as a gateway to a world of public APIs.
You can also filter the APIs by category and find the one that's right for you üñê

## sweetalert2

<a href="https://github.com/sweetalert2/sweetalert2" class="link">sweetalert2</a> is a beautiful, responsive, customizable, accessible replacement for JavaScript's popup boxes without external dependencies.
Primarily designed for JavaScript projects, you can integrate it with the three big frontend frameworks: Angular, React, and Vue.

## Video.js

<a href="https://github.com/videojs/video.js" class="link">Video.js</a> is a web video player built that supports HTML video and Media Source Extensions, as well as other playback techs like YouTube and Vimeo. It supports video playback on desktops and mobile devices.
As with sweetalert2, you can integrate this library with the three JavaScript frameworks: Angular, React, and Vue.
