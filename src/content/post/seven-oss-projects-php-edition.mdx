## Nextcloud

<a href="https://github.com/nextcloud/server" class="link"> Nextcloud</a> is a suite of client-server software for creating and using file hosting services.
It is an open source alternative to Google Suite. It can be hosted in the cloud or
on-premises ☁️

## Matomo

<a href="https://github.com/matomo-org/matomo" class="link">  Matomo</a> is an open source alternative to Google Analytics that protects your data and your customers' privacy.
Matomo lets you easily collect data from websites & apps and visualise this data and extract insights 📈

## Flarum

<a href="https://github.com/flarum/flarum" class="link"> Flarum</a> is a delightfully simple discussion platform for your website: it's community framework for small to enterprise communities that is extensible, scalable and completely free💲

## Firefly III

<a href="https://github.com/firefly-iii/firefly-iii" class="link"> Firefly III</a> is a self-hosted manager for your personal finances. It can help you keep track of your expenses and income, so you can spend less and save more. It supports the use of budgets, categories and tags ❤️‍🔥

## LinkFree Generator

<a href="https://github.com/chriskthomas/linkfree-generator" class="link"> LinkFree Generator</a> is an open source alternative to Linktree, a tool to help you share everything you are, in one simple link.
It's simple to use and free! 🌲

## Infection

<a href="https://github.com/infection/infection" class="link"> Infection</a> is a PHP mutation testing library based on AST (Abstract Syntax Tree) mutations. It works as a CLI tool and can be executed from your project’s root.
Mutation Testing is a fault-based testing technique which provides a testing criterion called the Mutation Score Indicator 🐉

## OpenCart

<a href="https://github.com/opencart/opencart" class="link"> OpenCart</a> is a free open source e-commerce platform for online merchants. OpenCart provides a professional and reliable foundation from which to build a successful online store.
